to launch this project you need : 
 1. Download  mongoDb ( link: https://www.mongodb.com/download-center/community) 
 2. Create mongodb folder
 3. put downloaded mongoDb inside this new folder
 2. run-mongo.bat 
 3. run nodemon server.js and go to http://localhost:4000/users/MVC/login to use MVC app 
 4. go to client folder and run npm start to run vue js app, app will be running on http://localhost:8080
